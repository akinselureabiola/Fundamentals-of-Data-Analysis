# Customer Churn Prediction
A machine learning project that predicts customer churn using classification models.

- Data cleaned and preprocessed
- Trained multiple models
- Evaluated using accuracy, precision, recall, and F1
- Compared results to find the best performer
- Models evaluated include Logistic Regression, Random Forest, Decision Tree, SVM, and KNN.
